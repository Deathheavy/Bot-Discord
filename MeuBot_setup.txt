Pré-requisitos:
Node.js instalado: https://nodejs.org/pt
Conta no Discord

1. Crie o bot no Discord Developer Portal
Acesse: https://discord.com/developers/applications
Clique em "New Application" e dê um nome
Vá em "Bot" → "Add Bot"
Marque a opção:

Message Content Intent

!!!!Copie o Token do Bot e salve para depois!!!!

2. Dê permissões ao bot de ver mensagens, enviar mensagens e ver historico:

Vá em OAuth2 → URL Generator
Marque:

Scopes: bot

Bot Permissions: Send Messages, Read Messages e View Channel

Copie a URL gerada e acesse no navegador
Adicione o bot no seu servidor do Discord

3. Estrutura do projeto
Crie uma pasta do projeto depois abra o cmd como administrador na pasta criada e execute os seguintes comandos:


npm init -y
npm install discord.js
npm install dotenv

/PROJETO
├── index.js
├── .env
├── package.json
├── package-lock.json     
└── node_modules

4. .env — Crie um arquivo na pasta do seu bot sem nome e no formato .env e coloque seu token que você copiou no inicio

DISCORD_TOKEN=SEUTOKENAQUI4423424252525252

5. index.js — Código principal do bot
   .env — local onde o seu token fica protegido


6. Antes de executar
Adicionar o token do seu bot em .env
Abrir o arquivo jogos.json e atualizar os nomes e links com o bloco de notas e sempre manter o formato exemplificado no arquivo
Atualizar o caminho do .bat com o bloco de notas para o local do seu bot caso for utilizar e mudar o formato de .txt para .bat
Adicionar a id de um canal privado na linha 123 do index.js para que você possa adicionar jogos direto do Discord através do arquivo + nome


7. Rodar o bot(VERIFIQUE SE REALIZOU TODAS AS ETAPAS DE 1 A 6 ANTES DE RODAR!)
Abra o cmd na pasta do projeto e digite:

node index.js
